::: PROJECT :::

NIO Automotive - concept

::: CATEGORY :::

  • HMI Design|

  • UI Design|

  • UX Design|

  • Design Management

::: YEAR :::

2016 - 2019

::: SYNOPSIS :::

As the EV industry progresses so does the idea of next generation of interfaces and console screen experiences. An idea came of a revoluionary new screen layout that would bring something new to the market. A bigger interaction zone that would connect the driver and copassengers. When it comes to thinking in terms of the future this would be a perfect task. Leading a small team made this concept possible and the result of what could be functional one day left everybody interested in user behaviors of tomorrow.

01 :::

The empathy

The infotainment system of tomorrow needs to evolve and bring new ways of interaction to the driving experience. What is the next step? Ideally it would be a bigger interaction surface by creating a connection between driver and co passenger to use the same surface in harmony. But how would the users behave in this kind of shared experience. Do they need it?

02 :::

The definition

We try to identify the core problems of a smaller console screen surface and try to see what would make the passengers work together. We imagine a bigger screen with more information avaiable for all the passengers and certain ways of everybody being able to interact with it. Brainstorm sessions needed the box to be wide open and innovation ideas to start flooding in.

03 :::

Ideation

We start generating different ideas ideas and quickly the picture start forming of something so interesting that nobody knows how to approach it completely. Only through trial and error we push forward to the next phase of the process. Innovation leaves so much room for new ideas that it sometimes becomes difficult to set a frame where the creativity needs to stop.

04 :::

Prototype

This requires different prototype tests to see how actually users behave while using the interaction spectre and new layout dimensions that are given to them. Needles to say it leaves designers with moments of doubts how to solve certain issues.
